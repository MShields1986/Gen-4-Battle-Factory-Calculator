# Gen 4 Battle Factory Calculator

Another damage calculator and team analyser for the Gen 4 Battle Factory (Platinum / HeartGold / SoulSilver).

## Quickstart

**Requirements:** Docker and Docker Compose.

```bash
# Clone and start
git clone https://github.com/MShields1986/Gen-4-Battle-Factory-Calculator
cd Gen-4-Battle-Factory-Calculator
docker compose up -d
```
Open a browser and open http://localhost:8080

To stop the server:

```bash
docker compose down
```
